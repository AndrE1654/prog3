# Завдання-Unit-тестування програм

**Умова:**
1. Встановіть необхідні залежності для написання unit-тестів:<br>
a. pip install pytest pytest-cov<br>
2. Використовуючи одне з завдань з ПЗ (ООП) написати unit-тести згідно наступних вимог:<br>
a. Файл з unit-тестами повинен мати назву test_<назва програми>.py<br>
b. Програма, що покривається тестами повинна мати не менше 3 функцій/методів<br>
c. Кожна функція/метод повинна мати не менше 2 тестів <br>
d. Покриття тестами повинно бути не менше 85%. Для перевірки покриття тестами використовуйте наступну команду:<br>
 pytest --cov=<назва програми> <br>
e. Сформуйте html-звіт про покриття програми тестами<br>
pytest --cov=<назва програми> --cov-report html<br>
Завантажити виконане завдання та тести на персональний github<br>
