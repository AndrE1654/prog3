# Завдання-Unit-тестування програм

**Умова:**
1. Встановіть необхідні залежності для написання unit-тестів:
a. pip install pytest pytest-cov
2. Використовуючи одне з завдань з ПЗ (ООП) написати unit-тести згідно наступних вимог:
a. Файл з unit-тестами повинен мати назву test_<назва програми>.py
b. Програма, що покривається тестами повинна мати не менше 3 функцій/методів
c. Кожна функція/метод повинна мати не менше 2 тестів 
d. Покриття тестами повинно бути не менше 85%. Для перевірки покриття тестами використовуйте наступну команду:
 pytest --cov=<назва програми> 
e. Сформуйте html-звіт про покриття програми тестами
pytest --cov=<назва програми> --cov-report html
Завантажити виконане завдання та тести на персональний github
